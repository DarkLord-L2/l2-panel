Общий доступ к иконкам

Ваши картинки публикуются в публичный GitHub Gist. Любой, кто откроет ссылку на эту панель, увидит их автоматически — но добавлять и менять их может только тот, у кого есть токен.

Токен (только на этом устройстве)

Создаётся на github.com → Settings → Developer settings → Personal access tokens (classic), scope «gist». Никогда никому не пересылайте его.

Как пользоваться панелью

Персонажи

Сверху выбираете расу и класс. У каждого класса — своя панель и своя библиотека иконок: переключаетесь между персонажами, и всё остаётся на своих местах.

Иконки и группы
  • Меню ☰ → «Добавить иконки»: выбираете «Для одного класса» или «Общая для всех классов», выбираете группу (существующую или создаёте новую по имени) и жмёте «Выбрать файлы…».
  • Группа «для одного класса» видна только на панели этого класса. Группа «общая для всех» появляется у каждого персонажа автоматически, без повторной загрузки.
  • Иконки показываются полоской из подписанных групп под панелью — у общих групп рядом с названием значок «общая».
  • Из группы перетащите иконку на любую ячейку — она там останется. Одну иконку можно ставить в несколько ячеек.
  • Файл из проводника можно бросить и сразу на ячейку — он попадёт и туда, и в группу «Иконки» текущего класса.
  • Иконку можно перетащить с одной ячейки на другую: пустая ячейка её примет, а если там уже есть иконка — они поменяются местами.
  • Перетащите иконку с ячейки в пустое место страницы — она уберётся с панели (в группе останется).
  • Крестик на иконке убирает её из группы, крестик у названия группы — удаляет всю группу (на уже расставленные ячейки это не влияет). Эти крестики, как и само добавление иконок, доступны только владельцу — тому, у кого в этом браузере сохранён токен из «🌐 Общий доступ». У остальных их нет вообще.
  • У владельца внизу колонки с иконками есть кнопка 🗑 — по умолчанию крестики удаления скрыты, чтобы не мешали смотреть иконки; кнопка их показывает и прячет обратно.
  • Кнопка «‹» вверху колонки сворачивает её в узкую полоску (нажмите «›», чтобы развернуть обратно) — иконки при этом никуда не пропадают, просто скрыты из вида. Браузер запоминает, свёрнута она или нет.
Макросы
  • ☰ → «➕ Добавить макрос» — доступно каждому, кто открыл панель, и хранится только в его браузере.
  • Иконку для макроса выбираете из набора, который выдаёт владелец панели (кнопка «+ Добавить иконку для макросов» пополняет этот набор — общим для всех он станет, только если владелец опубликует его через «🌐 Общий доступ»).
  • В самом макросе — до 12 строк текста, в каждой до 25 символов.
  • Готовые макросы показываются полоской «Мои макросы» под иконками — перетаскиваются на ячейки панели так же, как обычные иконки.
  • Клик по ячейке с макросом (не перетаскивание) показывает его текст.
Клавиши
  • Клик по бейджу в углу ячейки — панель ждёт нажатия. Нажмите любую клавишу (буква, цифра, F1–F12…), и она привяжется.
  • Нажмите ту же клавишу ещё раз на той же ячейке — бинд снимется, бейдж снова пустой.
  • Esc — отменить выбор клавиши.
Ряды и строки
  • Всего у класса 10 строк по 12 ячеек, на экране видно до 6 рядов.
  • Кнопки «двойные стрелки» слева от нижнего ряда открывают ряд выше и скрывают верхний.
  • Маленькие стрелки ▲/▼ с номером слева от каждого ряда перелистывают этот ряд по строкам 1–10 (после 10 идёт снова 1). Сам ряд остаётся на месте, соседние ряды не меняются.
Доп. панели 11–13
  • Кнопка « справа от панели добавляет плавающие панели — строки 11, 12 и 13, по одной за нажатие. Когда открыты все три, следующее нажатие убирает их.
  • Перетаскивайте их за ручку ≣ куда угодно — за край экрана они не уйдут.
  • Кнопка ▦ на панели меняет форму: 1×12 → 2×6 → 3×4 → 4×3 → 6×2 → 12×1.
  • Иконки и клавиши на них вешаются так же, как на основной панели. Перелистать эти строки нельзя.
Перенос, поворот и замок
  • Ручка ≣ в столбике кнопок — перетащить основную панель в любое место экрана. Двойной клик по ручке возвращает её на стандартное место.
  • Кнопка ⇄ ставит основную панель вертикально и обратно.
  • Кнопка 🔓/🔒 — замок: пока он закрыт, ничего нельзя добавить или убрать (иконки, клавиши, очистка). Перелистывание и перетаскивание панелей работают.
Очистка

Правый клик по ячейке — убрать картинку, убрать клавишу или очистить ячейку целиком.

Поделиться (одна ссылка, один класс)
  • Кнопка «Поделиться» снизу слева копирует ссылку, внутри которой зашита вся раскладка текущего класса — картинки, клавиши и ряды.
  • Получатель открывает ссылку, подтверждает — и раскладка загружается ему в тот же класс, а картинки попадают в его библиотеку.
  • Чем больше картинок на панели, тем длиннее ссылка. Если мессенджер обрезает её — передайте раскладку файлом через ☰ Экспорт/Импорт.
Общий доступ (GitHub) — только для владельца
  • Это для сценария «я один раз добавил иконки — сто друзей открыли ссылку и сразу всё увидели», без ручного импорта.
  • Картинки хранятся в публичном GitHub Gist. Токен для записи храните только вы, в своём браузере — никто другой не сможет ничего опубликовать, даже открыв ту же ссылку.
  • Вставьте свой токен один раз, нажмите «📤 Опубликовать все классы» — публикуются и сами группы иконок (личные для класса и общие для всех), и картинки, уже расставленные по ячейкам панели.
  • У любого, кто впервые открывает ссылку на панель, группы и картинки подтянутся сами. «🔄 Обновить из общего доступа» — подтянуть свежую версию вручную (например, если вы что-то добавили после того, как друг уже открывал панель).
  • Бинды клавиш не публикуются — каждый выставляет их под свою клавиатуру сам.
  • Набор иконок для макросов публикуется вместе с остальным — так у всех будет одинаковый выбор картинок для своих макросов. Сами макросы (текст) при этом не публикуются, они личные.
  • Пункт «Добавить иконки» в меню ☰ появляется только после того, как токен сохранён в этом браузере (через «🌐 Общий доступ» — этот пункт виден всем, чтобы было куда вставить токен).
  • Токен привязан к конкретному сайту (домену) в браузере — если открываете панель по разным ссылкам (например, старую версию и новую на GitHub Pages), токен придётся вставить заново на каждой.
Меню ☰ (сверху слева)

«Добавить иконки» появляется только у владельца (после ввода токена). «🌐 Общий доступ» открыт всем — там же владелец вставляет токен. Экспорт всех классов в один файл, импорт из файла и очистка панели текущего класса — доступны всем. Экспорт нужен для бэкапа или переноса на другой компьютер — в остальном всё сохраняется автоматически в этом браузере.
