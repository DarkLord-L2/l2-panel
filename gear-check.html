Проверка буста

Средний процент буста: —

В последней переписи нет участников — сначала заполните «Перепись клана».

Как пользоваться проверкой буста

Таблица

Участники берутся из последней «Переписи клана». Столбцы — пункты буста (Бафф, Зілля и т.д.), у каждого свой вес (число-бейдж в шапке). «Общие баллы» — сумма веса включённых пунктов, «Процент буста» — эти баллы от максимума по всем пунктам.

Карточка участника (клик по строке)

Открывает панель справа: тумблеры включают/выключают пункты у этого участника. Баллы и процент пересчитываются сразу.

Класс и аватарка участника

Кнопка «⋮» справа от ника (у админа) открывает окно, где можно вписать класс — он показывается подписью под ником — и загрузить аватарку.

Настройка столбцов (главный админ и админ)

Кнопка «+» в шапке таблицы, последним столбцом, добавляет новый пункт: имя, вес, потом можно загрузить картинку. Если разделов ещё нет, автоматически создастся раздел «Общая»; если разделов несколько — спросит, в какой добавить.

То же самое можно делать в панели справа: внизу кнопка «✎ Редактировать» показывает у разделов и пунктов кружок (скрыть/показать), карандаш (имя/вес/фото) и крестик (удалить совсем); «+ Раздел» тоже внизу. Пока «Редактировать» не нажат, панель чистая — видны только тумблеры.

Скрыть — пункт пропадает из таблицы и не считается в баллах, но тумблеры участников по нему сохраняются: в любой момент можно снова показать, ничего не переделывая. Удалить — совсем, без возврата.

Пункт

Пока отмечено — поле «Подпись под названием» выше не используется, показывается значение из «Значения по классам» для класса участника.

Пункты с одинаковым подразделом группируются под общей подписью, отдельно от остальных пунктов раздела.

Значения по классам

Для пунктов с отметкой «Своя подпись для каждого класса»: у баффа — свой код, у тату — свои статы.
